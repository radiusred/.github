# Contributing

We welcome public contributions to Radius Red's public profile repos.

Getting started:
- Fork the repository and create a feature branch.
- Install dependencies and run tests locally if applicable.
- Ensure changes are small, well-scoped, and documented in PR descriptions.
- Open a PR against main; rebase rather than merge commits.
- Provide a brief rationale and expected impact in the PR description.
- Include tests or updates to tests if the change affects behavior.
- Do not include secrets or internal data in public PRs.

Documentation alignment:
- Update public docs with exact references to shipped code.
- Reference merged PRs or docs for traceability.

If unsure, open an issue describing the goal and request guidance before implementing.
