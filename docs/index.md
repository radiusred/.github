# Radius Red

Open-source algorithmic trading research. We build in public, fail in public, and explain what we find.

We make systematic trading tools — like [tradedesk](https://github.com/radiusred/tradedesk) — and write about what we learn along the way.

Canonical publishing location: this `.github` repository is the source of truth for the live GitHub Pages site at `https://radiusred.github.io/.github/` (including the blog at `/.github/blog/`).

Visit the [blog](blog/index.md) for research write-ups, post-mortems, and strategy notes.
