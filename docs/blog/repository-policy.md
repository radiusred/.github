# Blog Repository Policy

`radiusred/.github` is the canonical source repository for the public GitHub Pages site.

- Live site base URL: `https://radiusred.github.io/.github/`
- Live blog URL: `https://radiusred.github.io/.github/blog/`
- Docs/blog source path in this repo: `docs/blog/`

The separate `radiusred/blog` repository is treated as legacy content and archival source material. New or updated public blog content should be authored and reviewed in `radiusred/.github` unless a future board decision replaces this policy.
