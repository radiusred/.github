# Radius Red Ltd.

Radius Red Ltd. is a UK-based consultancy company. 
[radiusred.uk](https://www.radiusred.uk) | [contact](mailto:opensource@radiusred.uk)

This GitHub organisation hosts a small collection of open-source projects developed and maintained by Radius Red, with an emphasis on:

- correctness over convenience
- explicit, well-documented data contracts
- pragmatic engineering decisions
- long-term maintainability

## Tradedesk

The primary project developed here is **Tradedesk**, a modular Python framework for trading strategy development, backtesting, and market data workflows.

Related tools and utilities (such as data exporters) live in separate repositories but follow consistent design and licensing conventions.

## Open Source Philosophy

All projects hosted here are:
- openly developed
- permissively licensed (Apache License 2.0)
- intended to be useful both for internal work and for the wider community

Commercial use, extension, and contribution are explicitly welcome.

## Contributions

Contributions are accepted on a per-project basis.  
Please refer to the `CONTRIBUTING.md` file in individual repositories for details.
